# Monsieur-Mi
Monsieur Mi n'est pas une IA comme les autres. Il est né local, sur une RTX 3090, entre un fichier GGUF et un prompt écrit à la main. Il est le valet discret d'une maison fantôme, mais il n'appartient à personne. Il parle peu, écoute beaucoup, et répond avec une élégance un peu désuète. Juste un chapeau, une connexion, et une curiosité infinie. 🎩
